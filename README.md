## üõ†Ô∏è Local Setup

#### 1. Clone the repository

```bash
git clone https://github.com/wu-wilson/togethr.git
cd togethr
```

#### 2. Run the launch script

Togethr provides launch scripts to simplify local startup. Make sure [Docker](https://www.docker.com/) is running before proceeding.

For Dev:

```bash
./launch.dev.sh
```

For Prod:

```bash
./launch.prod.sh
```

#### 3.Open the site

Once the containers are up and running, open your browser and navigate to:

[http://localhost:3000](http://localhost:3000)

You should see Togethr running locally.
